# Security Fundamentals

Notes and hands-on exercises covering the core concepts, principles, and controls that form the foundation of cybersecurity.

## Topics
- CIA Triad
- Authentication and Authorization
- Identity and Access Management (IAM)
- Encryption and Hashing
- Risk Management
- Security Controls
- Security Operations
- Incident Response
- Logging and Monitoring
- Vulnerability Management
- Security Policies and Procedures

## Goal
Build a strong understanding of why security controls exist, how they work, and how they reduce organizational risk.

> Learn the principles first. Tools and technologies will change, but fundamentals remain.
